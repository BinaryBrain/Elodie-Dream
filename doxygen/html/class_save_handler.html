Detailed Description

The class used to manage saves. There should be only one SaveHandler. To create a save file, set the path with setPath(), add the things you want with the add() functions and use save() when you're done.

To load a save, set the path of the file, use load() and then be sure to read what you added in the right order (when you saved).

At the moment, the buffer is a simple string and the add functions push the data on a new line each time.
